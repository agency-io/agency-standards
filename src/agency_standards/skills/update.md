Update generated architecture tests in this project.

## Project context
!`agency-standards context`

## Active standards
!`agency-standards standards prompts`

## Instructions

1. Find all files in `tests/architecture/` whose first line contains `# GENERATED by agency-standards`.
2. For each file:
   a. Read the standard ID from the header: `# GENERATED by agency-standards | standard: <id>`
   b. If `$ARGUMENTS` is set and does not match this ID, skip it.
   c. Look up that standard in the active standards list above. If not found, skip it.
   d. Regenerate the test content for that standard using its prompt and the current project context.
   e. Preserve everything after `# --- CUSTOM (preserved on update) ---` from the **original** file verbatim.
   f. Write the updated file, keeping the same two-line header format.
3. Report which files were updated and which were skipped.
